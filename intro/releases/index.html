V2.0.4 (Mar 30 , 2021)#

New Features#

  • ValQ Sign-in Methods: ValQ allows 2-way sign-in for the users. Users can log-in with Native username/password managed by ValQ apart from Microsoft Office 365 sign-in.

Enhancements#

  • Lengthy Node name is truncated in the Tree view to avoid overlapping on other values.

Bug Fixes#

  • The Quick Editor does not show Link to Model drop-down option in the Node Type column.
  • Node search is fixed in the Table view.
  • The node level and tool tip both display 01, when you use the Export to Excel and Import from Excel features in Quick Editor.
  • Empty or 0 Secondary KPI values do not show error.
  • Fixed the Node Display in tree view. When you select any Value Display scale ( 0, 0k, 0.0m, 0.0b ) in the Navigation panel field, the value of the node does not overlaps.
  • Secondary KPI 0 values do not disappear when you switch on to a sub model or model merged node.
  • Power BI filters interact with ValQ for Node value and Series switcher properly
  • The View Allocation History is grayed out when we get the confirmation popup for Reset All option.
  • Prefix and suffix gets displayed now even when the value of the node is 0.
  • In the Node Configuration > Value Range min and max fields and Model > Settings > Conditional Formatting threshold sections, you can now enter 1/100th value and lesser.
  • All the leaf nodes are exported in the Export as Datasource Tree structure, when the option Export only leaf nodes is selected.
  • If a model is shared, secured nodes and sub models will not be visible for other user(s) upon searching across all the screens.
  • When the simulation months is set as Jan to Dec then the tooltip of YtD in the Value Display section shows the three letter abbreviation that you set in the Time aggregation Labels section.

New Features#

  • WritebackÂ Scenario feature is also available in the Simulate > Compare Scenarios screen.

Enhancements#

V2.0.2#

New Features#

  • Node limit has been lifted on all the plans. Now models can be built for unlimited nodes.
  • Upgrade pop-up has been added to the visual on PBI desktop and Service for non-enterprise plans.

V2.0.0 (Jan 20 , 2021)#

New Features#

  • User Security feature in ValQ Collaborate to grant specific access to the Model/nodes, Scenarios, and Planning Series to user(s) and group(s).
  • Sensitivity Analysis of the effect of varied input KPIs on a selected target node. You can select the input nodes that are to be simulated and percentage of variance. Together the 2 parameters determine the result on the output node in the form of a Tornado chart.
  • Digital Board room experience with Touch support. ValQ features are now automatically optimized for Touch Support on touch-enabled devices for both Canvas and Tree view.
  • The Plan tab includes Writeback feature that triggers to send data to an external API.
  • You can enable to show the Secondary KPIs in the Table view.

Enhancements#

  • Planning series implements single lock icon as a toggle to Hold/Release the cells.
  • Plan/Simulation/Variance Analysis tab introduces Size columns to fit option to auto-size the columns width.
  • Decimal places on node gets diplayed in theÂ Plan input editorÂ based on the node's configuration.
  • Number formatting for Trend percentage now shows the selected sign in Tree & Table View.
  • Added Show Composite Nodes as a toggle in Navigation Panel for models which have Composite Nodes.
  • The Manage Columns settings in the Model > Node > Quick Editor is grouped similar to the labels in the Edit Node configurations.
  • The Export as datasource feature includes Export all option. This option allows you to export all the saved series/scenarios in the model including future updates.

Bug Fixes#

  • The selections made in the Variance Analysis tab are now saved and retained even when you quit the Analyze tab.
  • The value of Node completely gets diplayed in the Plan Grid (Decimal points) irrespective of the node's configuration.
  • Fixed the "Tab" button functionality in the Planning Weight feature. The use of the "Tab" button to move to the next period is disabled for the last period and the use of the "Shift + Tab" to come back to the pervious period is disabled for the 1st period.
  • Refer the ValQ 2.0 Connected and Collaborative Planning Features blog.

V1.8.3 (Jul 30,2020)#

Features#

  • Added feature Rolling Planning to automatically roll out 12 months forecast over a set time frame.
  • Added feature to export your model as a web data source with the option Export as datasource.
  • Table view can now be customized to edit the columns and get summarized views.

  • Model Merge enabled for a selected node from the sub-model to another node in the workspace.

Enhancements#

  • Made performance based Member Console changes
  • NotesÂ feature is now replaced asÂ Comments.

  • Improved the Writeback data JSON format.

V1.7.0 (May 12,2020)#

Features#

  • Added Model Merge feature that allows users to merge ValQ models from different users within the workspace.
  • Added Time Series Forecasting feature that allows you make a forecast based on past data.
  • You can manage the Users/Groups, and Models from your Member/ValQ Console account.

Enhancements#

  • Enabled Scenario Collaboration (ValQ - Professional, Enterprise) for Private and Shared Scenarios with view/edit access.
  • Enabled Read-only Mode in Power BI Service, with read-only access for Visualization (Table/Tree mode), Nav Panel, Export, Compare, and Variance Analysis like features.
