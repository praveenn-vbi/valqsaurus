## 1. Navigation Panel

Turn on/off the toggle under **Enable Navigation Panel** to display/hide the <Link to={useBaseUrl("simulation/navigation_panel")}>Navigation Panel</Link> on the *Simulate* tab.
Click on the editor(Pencil) icon to edit the name of the property.
To hide a property click the eye icon against the property you wish to hide.

<div style={{ textAlign: "justify" }}>

## 4. Number Formatting

You can configure the following properties with Number Formatting settings:

1.  Under Scale Suffix, set the property Thousands to "k"
2.  Set the property Millions to "m"
3.  Set the property Billions to "b"
4.  Set the property Trillions to "t"

<div style={{ textAlign: "center" }}>
  <Zoom>
    <img alt="Number Formatting" src={useBaseUrl("/doc-images/model/s18.png")} />
  </Zoom>
</div>

5.  Under Display Formatting, set the property Display zeroes as
    to the option "0"
6.  Set the property Display Negative values as to the option "(0)"
7.  Set the property Decimal Separator to "."
8.  Set the property Thousands Separator to ","

<div style={{ textAlign: "center" }}>
