## Step-by-Step Guide

In this section, you will learn how to visualize sales data using ValQ in less than 5 minutes. To do this you will have to

1. [Import data from Microsoft Excel](#step-by-step-guide) (or any other data source)
2. [Assign your data fields to ValQ](#2-assign-your-data-fields-to-valq-br-)

You have built your first visualization using ValQ. Take some time to explore this further.

ValQ also packages a few inbuilt examples. To view them, click on the ‘Model’ tab on the top.
Select ‘Open a Sample Model’. You will see several inbuilt examples.
Click on any model, say ‘Income Statement’, and the model will load. To explore more such models, visit <a href="https://valq.com/demos" target="_blank">ValQ-Demos</a>.
