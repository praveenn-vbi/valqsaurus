An `IF` statement returns a value depending on whether the condition that is being evaluated is TRUE or FALSE. 

## Syntax
```javascript
IF( logical_test, value_if_true, value_if_false )
```
## Arguments
`logical_test` – The condition to be evaluated.

`value_if_true` – The value to be returned if the condition is met.

`value_if_false` – The value to be returned if the condition is not met.

## Return Value
Either `value_if_true` or `value_if_false`.

## Remarks
- Returns Blank when the formula is misspelled.
- The `IF` function returns Error in any of the following cases :
    - value_if_false argument is missing
    - The argument value_if_true or value_if_false are not of numeric data type
    - Not using appropriate <Link to={useBaseUrl('references/formula-list#other-operators')}>operator</Link> in logical_test

## Example
Consider the Revenue and Cost of Goods Sold nodes to follow the trend as shown below

| Parameter          | Jan | Feb | Mar | Apr | May | Jun | Jul | Aug | 
|--------------------|-----------|-----------|-----|-----|-----|-----|
| Revenue            | 100 | 124 | 137 | 0   | 23  | 45  | 52  | 36  | 
| Cost of Goods Sold | 25  | 25  | 25  | 25  | 25  | 25  | 25  | 25  |

<div style={{textAlign: 'justify'}}> 

Now, a new node Gross Profit Margin can be computed using the `IF` function.
Gross Profit Margin is calculated as (Revenue-Cost of Goods Sold)/Revenue.
In case, Revenue has period value of 0, Gross Profit Margin should return 0.
</div> 

To achieve this, the formula should be 
```javascript
IF(Revenue==0,0,(Revenue-Cost of Goods Sold)/Revenue)
```
The result of the above formula is shown below

| Parameter          | Jan | Feb | Mar | Apr | May | Jun | Jul | Aug |
|--------------------|-----------|-----------|-----|-----|-----|-----|
| Gross Profit Margin| 75% | 80% | 82% | 0   | (9%)| 44% | 52% | 31% |
