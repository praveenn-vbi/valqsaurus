Plan Features

Let's take a look at the features of the Plan tab in this section.

#1. Plan Table

You can see the name of the plan you are currently working on as highlighted in the figure below. The footer highlights the series on which the plan has been created.


#2. Add New Series

Click on the '+ New Series' to add another series to the Plan tab.


#3. Search for a Node

You can take a look at a particular node and it's children by entering the node name or ID using the search bar.


#4. Get Insight on a Value

On clicking any cell with value, the number format and scaling that were configured while modelling are retained. You can see the absolute value on the 'Number Bar' and the Calculation Method and Time Aggregation on the footer as highlighted in the below image.


#5. Action Menu

In the 'Action Menu’, you will find various options such as

#5.a) Weight

Weight distributes a node's total value based on the period values of a node choosen by the user.


#5.b) Equal

Equal distributes a node's total value equally over the period.


#5.c) Trend

Trend applies the change based on a node's period value to the rest of the year.


#5.d) Copy-to

Copy-to is used to copy a node's period value and apply the same to the rest of the year.


#5.e) Auto Forecast

Auto Forecast enables the user to apply Time Series Forecasting methods to the plan. Click here to know more about it.


#5.f) Allocate Children

Child nodes are allocated either by 'Weight' or 'Equal' under this option and 'Copy-to' is used to copy a node's value to its sibling nodes.


#5.g) Hold and Release

Hold and Release can be used to lock or unlock certain cells as needed.


#5.h) Note

Note helps the user to add, modify or delete notes.


#5.i) Refresh Datasource

On clicking Refresh Datasource, the newly created series gets updated if there are any changes to the input data.


#5.j) View History

View History is used to view the changes performed and also roll back changes when needed. 'Reset all' would delete all the allocation adjustments made to the series. 'Restore to this step' traces back to the selected step.
