Plan Features

Let's take a look at the features of the Plan tab in this section.

#1. View current plan table

You can see the name of the series you are currently working on as highlighted in the figure below. The footer highlights the base series on which the plan has been created.


By clicking on the settings icon, the newly created series can be set as primary or comparison series for the Plan tab. You can also rename or delete the series here.


#2. Add New Series

Click on the '+ New Series' to add another series to the Plan tab.


#3. Search for a node

You can take a look at a particular node and it's children by entering the node name or ID using the search bar. The model along with the all and period values get displayed in a table form. In case, the empty series has been chosen as the series to start with, only the structure and relationships are displayed and not the values.


#4. Get insight on a value

On clicking any cell with value, the number format and scaling that were configured while modelling are retained. You can see the absolute value on the 'Value Bar' and the Calculation Method and Time Aggregation on the footer as highlighted in the below image.

Note
Just like a formula bar in spreadsheets, you can perform basic operations on a value in the value bar


#5. Action Menu

In the 'Action Menu’, you will find various options such as follows


#5.a) Weight

You can edit the original total value - manually or by simply, adding or subtracting to get new total value. Weight distributes a node's new total value based on another node choosen by the user. Weight has options to choose series and node by the user. With change action option, the user can choose to Overwrite Values or Append Values.


Total Value : The node values under the Total column

Period Value : The node values under the period (month) columns

Overwrite Values : The original value is overwritten by the new value and this new value gets distributed to the spread on top of the original value

Append Values : The original value is appended by the change (i.e. the difference between the original value and new value) and the change gets distributed to the spread by appendeing to the orignal values of the spread

#5.b) Equals


Like Weight, here you can edit the original total value - manually or by simply, adding or subtracting to get new total value. Equal distributes a node's total value equally over the periods. With change action option, the user can choose to Overwrite Values or Append Values.

#5.c) Trend

Trend enables the user to increase/decrease the original period values - By Percentage or By Value and then apply the change to the remaining periods of that node. With change action option, the user can choose to Overwrite Values or Append Values.


Note
Increase Trend option accepts both positive and negative values.

#5.d) Copy-to

Copy-to option takes a node's period value and applies the same value to the remaining periods of that node. The change gets updated to the parent node as well and is highlighted in yellow as seen below.


#5.e) Auto Forecast

Auto Forecast enables the user to apply Time Series Forecasting methods to the plan. Click here to know more about it.


#5.f) Allocate Children

Child nodes are allocated either by 'Weight' or 'Equal' under this option and 'Copy-to' is used to copy a node's value to its sibling nodes.


#5.g) Hold and Release

Hold and Release can be used to lock or unlock cells as needed. The feature can be applied on cell, period, row or column level. This option helps you to configure rest of the plan tab and then release the cells kept on hold.


#5.h) Add a comment

Note helps the user to add, modify or delete comments on cell level.


#5.i) Refresh Datasource

On clicking Refresh Datasource, the newly created series gets updated if there are any changes to the input data.


#5.j) View Allocation History

View Allocation History is used to view the changes performed and also roll back changes when needed. 'Reset all' would delete all the allocation adjustments made to the series. 'Restore to this step' traces back to the selected step.
