Sample Business Cases

#Scenario 1: To allocate the parent node’s changes to some of the immediate child nodes

1. For our example, expand East node. Now select the node 'Industrials' and lock the node (see Figure below).


2. Now select the Total Cell for node 'East' as shown in the below Figure. You can update the number by double-clicking the cell and entering the number directly in the value bar. Or you can enter the number in the popup.


3. Now double click the total cell and type '+10%' at the end of the number and click on 'Weight' as shown in the below Figure.


  1. You will be able to view the 'Spread by Weights' screen.

5. As a result of Scenario 1, you will be able to view the Input Editor screen updated with the configured values (see Figure below). The cells which were impacted because of the allocation operations turns yellow.


#Scenario 2: To allocate a node’s increase/decrease in aggregate value to some of the periods

1. For our example, select the Node 'Cost of Sales' for April month (see Figure below).


2. Increase the cell value by 10% and click Enter (see Figure below).


3. Now click the “Copy To” icon to copy the value from April across the periods till December (see Figure below)


4. After clicking on the Copy to icon option, the Input Editor screen looks similar to the Figure shown below.


#Scenario 3: To increase the period values of a node based on a monthly trend

1. For our example, select the Node 'Other Costs' for Jan month (see Figure below).


2. Now click the 'Trend' icon (see Figure below)


3. Now the Trend pop up opens. For our example, select the 'By Percentage' option and enter value 2 in the Increase Trend section. Now select the Overwrite Values option and click 'Apply' (see Figure below).


4. Based on the above configuration, you will be able to view the Input Editor screen as shown below.


#Scenario 4: To perform driver-based allocation

1. For our example, select the Node 'Support Costs' in the Total column (see Figure below).


2. Now click the 'Weight' icon (see Figure below)


3. The Spread by Weight pop up opens. For our example, select the Node as 'Cost of Sales'. Select the 'Overwrite Values' option and click 'Apply' (see Figure below).


4. As a result of Scenario 4, you will be able to view the Input Editor screen loaded with the redistributed values (see Figure below).


#Scenario 5: To increase the period values of a node according to a combination of trends

1. Select the Node 'Rent' and lock the entire Row.

2. Release the lock for the cells from Jan to Mar.

3. Now select the cell 'Jan'. Increase the value by 5% (see Figure below).


4. Now copy it across the cells. The Input Editor looks similar to the Figure as shown below.


5. Now release the lock for the complete Row of the Node Rent.

6. Select the cell 'Apr'. Increase the value by 10% (see Figure below).


7. Now copy it across the cells. The Input Editor looks similar to the Figure as shown below.
