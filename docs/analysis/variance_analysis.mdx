<div style={{textAlign: 'justify'}}>
Variance Analysis calculates variances against a benchmark for the KPIs to understand how it's performance impacts the business.
</div> <br />

<div style={{textAlign: 'justify'}}>
ValQ's Variance Analysis helps compare the values of two or more data series for each of the nodes in the model and 
calculate variances - absolute and percentage for all the nodes.
</ div> 

## Pre-requisite

A ValQ model with Comaprison is needed to perform Variance Analysis

## Access Variance Analysis 

To access Variance Analysis, click on the Variance Analysis option from the Analyze tab drop-down as shown below

## 10. Apply Filter to report

<div style={{textAlign: 'justify'}}>

Use the Filter option to analyze critical influencers.

<div style={{textAlign: 'center'}}>
  <Zoom>
<img alt="ValQ logo" src={useBaseUrl('/doc-images/analyze/filter.png')} />
  </Zoom>
</div>

There are two types of filters:

**Filter by Parent**: This allows you to analyze variance for a specific branch within the model. For example, you may be interested in analyzing variance for a particular location or product.

**Filter by Variance**: This allows you to identify top ‘n’ analysis for categories showing the most variances for a particular period chosen from the drop-down.
</div>

> **Note** <br /> You have the option to select whether you want to display only positive variances or only negative variances or both.

In this example, filter By Variance has been applied to filter the top 5 variances for Full Year.
