Variance Analysis

#1. Set Base Series and Compared Series

Variance Analysis helps compare the values of Base Series with that of the Compared Series for each of the nodes in the model and calculate variances for the nodes.

You can set the Base Series and Compared Series from the model that you want to compare here.

#2. Set a Scenario as Base Series

In the Analyze tab, you can implement the scenario that you have created in the Simulate tab as Base Series and compare it with Compared Series to perform Variance Analysis. This can be done by simply choosing the scenario that you want from the drop-down and clicking on the Apply scenario option.

#3. Choose a time frame to perform analysis

The Period drop-down allows you to specify a time frame for which you would like to run the analysis. Here, you can select multiple time frames like Full year, Quarter, Year-to-date, etc.

#5. Show Absolute & % Variance

In this section, you may choose which variances are important to you for analysis. Selecting these options adds them to the table. The positive variance items are shown in green and the negative ones in red.

#6. Show Hidden Nodes

You have an option to display the nodes that were hidden while building the model under the Model tab. Here, East is the hidden node and on selecting the Show hidden node option, it gets displayed. However, notice that the Overall Result doesn't get affected.


Note
Upon enabling show hidden node, the node stays hidden on the other tabs.

#7. Show/ Hide Unit column

The unit column is seen by default on the Analyze tab. To hide the column, click on the show unit to disable the feature.

#8. Apply Filter to report

Use the Filter option to analyze critical influencers. Two filters are available:

Filter by Parent: This allows you to analyze variance for a specific branch within the model. For example, you may be interested in analyzing variance for a particular location or product.

Filter by Variance: This allows you to identify top ânâ analysis for categories showing the most variances for a particular period chosen from the drop-down.

Note
You have the option to select whether you want to display only positive variances or only negative variances or both.

#9. Search for a node

You can search for a particular node by using the search bar as highlighted below. This feature can be used to identify a node when a model has a large number of nodes.
