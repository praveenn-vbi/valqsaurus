V1.7.0 - valQ (May 12,2020)

Earlier version: -
Next version: V1.8.3
Releases

Features#

Model Merge#

This feature allows users to merge models in valQ which can be created/managed by different users or groups within a workspace.

Advantages

  1. Break-down a complex model into sub-models which can be maintained independently
  2. Support for larger trees can be achieved with this feature
  3. Loading performance can be better than Power BI

Time Series Forecasting- Auto Forecast#

This feature allows to forecast for the series for a selected node and apply the values during planning



Enhancements#

Scenario Collaboration (valQ - Professional, Enterprise)#

Scenario Collaboration for Private and Shared Scenarios with View/Edit access

  • Private Scenarios - Scenarios that only you have access to, and not other members of the workspace

  • Shared Scenarios - Scenarios are made available to all the users of the workspace

    • View/Edit access Users can collaborate by creating and sharing scenarios with this feature.

    Some use cases:

    1. An user can create a private scenario and compare against a benchmark scenario created by a business user
    2. Business SMEs can create scenarios and share it with other users of the workspace and compare the drivers/outcomes.

    This collaboration features overcomes Power BI limitations:

    • Allows users to create and share scenarios in read/edit mode
    • Addresses the bookmark state issue

    Shared Scenarios
