V1.8.4 - valQ Collaborate (Aug 22,2020)#

Bug Fixes & Improvements#

  • Fixed Variance analysis & table view issues in Model merge reports

V1.8.3 - valQ Collaborate (Jul 30,2020)#

Features#

  • Added feature Rolling Planning to automatically roll out 12 months forecast over a set time frame.
  • Added feature to export your model as a web data source with the option Export as datasource.
  • New Table view - Table view can now be customized to edit the columns and get summarized views.
  • Added the Attribution Analysis which analysis the key attributes that contribute to the overall result simulation.
  • Model Merge enabled for a selected node from the sub-model to another node in the workspace.

Enhancements#

  • Made performance based Member Console changes
  • NotesÂ feature is now replaced asÂ Comments
  • Planning and Allocation history comments now available as consolidated comments.
  • Improved the Writeback data JSON format
  • Enhanced the Node configuration Formulas

V1.7.0 - valQ (May 12,2020)#

Features#

  1. Added Model Merge feature that allows users to merge ValQ models from different users within the workspace.
  2. Added Time Series Forecasting - Auto Forecast feature.
  3. The Member/ValQ Console account available for users/groups and models management in your workspace.

Enhancements#

  1. Enabled Scenario Collaboration (ValQ - Professional, Enterprise) for Private and Shared Scenarios with view/edit access.
  2. Enabled Read-only Mode in Power BI Service, with read-only access for Visualization (Table/Tree mode), Nav Panel, Export, Compare, and Variance Analysis like features.
  3. Added Model Details in the workspace, along with details like Series/Scenarios under Member/ValQ Management Console.
